# Simple Cybersecurity Recon & Port Scanner

Bu proje, siber güvenlik sızma testlerinin ilk aşaması olan **bilgi toplama (Reconnaissance)** ve **port tarama** işlemlerini otomatikleştirmek için Python ile geliştirilmiştir.

## Features (Özellikler)
- Hedef alan adının (Domain) IP adresini çözümler.
- En kritik 5 portu (21, 22, 80, 443, 3389) hızlıca tarar.
- Hafif, hızlı ve harici kütüphane gerektirmez.

## Usage (Kullanım)
Projeyi çalıştırmak için terminale şu komutu yazın:
```bash
python cyber_scanner.py
```
